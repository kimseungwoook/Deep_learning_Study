AlexNet
AlexNet은 LeNet-5의 합성곱 신경망 구조를 계승하였습니다. 하지만 대규모 데이터를 학습을 하기 위해서 하드웨어적인 부분과 알고리즘적 한계를 돌파한 현대 딥러닝의 시초입니다. 이 AlexNet의 특징은 총 5가지가 있습니다.

1. GPU 병렬 연산
   이 논문을 쓸 당시의 GPU는 GTX-580으로 대규모 데이터를 학습을 하기에는 Out of Memory 문제가 있었다. 따라서 네트워크를 반으로 쪼개어 2대의 GPU로 병렬 연산을 하였다.
   모든 layer에서 병렬 연산을 한 것이 아닌 특정 레이어(Conv3-layer, FC_layer)에서만 GPU간의 교차 통신을 하도록 하여 병목 현상을 방지하였다. 각 GPU로 교차 통신을 하지 않았다면 다론 모델 2개서 따로 학습을 하는 것과 동일 하므로 특정 layer에서만 교차 통신을 하였다.

2. ReLU활성화 함수 사용
   기존의 활성화 함수인 Sigmoid, tanh는 층을 깊이 쌓을수록 기울기가 소실되는 문제를 가지고 있었다. 하지만 ReLU는 음의 영역인 부분은 0으로 처리하고 양수인 부분은 y=x인 기울기를 사용하여 미분 값을 1을 유지하는 활성화 함수를 사용하였다.
   ReLU활성화 함수는 음의 영역인 부분의 뉴런은 기울기를 0으로 처리하므로 해당 뉴런에 대한 연산은 없게 된다. 따라서 양의 영역인 부분만 연산을 하므로 연산 속도가 이전보다 약 6배가 빨라졌다고 하였다.

3. LRN(Local Response Normalization)
   위에서 AlexNet은 ReLU 활성화 함수를 사용한다고 하였다. ReLU는 양수를 그대로 통과시키는 성질로 인해 특정 뉴런이 과도하게 활성화되기도 한다. 이러한 문제를 해결하기 위하여 3번째 이미지와 같이 주변 뉴런을 사용하여 과도한 특정 뉴런을 억제하도록 정규화를 하여 모델의 일반화 성능을 높였다.

4. 데이터 증강(Data Aumentation)
   거대한 파라미터 대비 부족한 데이터 수를 해결하기 위하여 이미지를 Random Crop과 좌우 반전등을 사용하여 데이터를 증강하였다.
   물리적인 파일 수는 고정이지만 Epoch마다 실시간으로 변형되어 입력이 되므로 Overfiting을 방지하였다.

5. DropOut
   4번 째 이미지를 보면 Dropout의 이미지이다. Dropout은 학습 시에 순방향 연산에서 50%의 뉴런을 끈다. 이로 인해 순방향 연산과 Backpropagation 연산에서도 해당 뉴런에 대한 연산은 제외된다. 이로 인해 뉴런들의 패턴을 모델이 통째로 외우는 현상(Overfiting)을 방지하였다.
