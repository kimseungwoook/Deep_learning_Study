하지만 CNN은 공간적, 지역적 특징을 얻어 MLP의 한계를 해결하였다.

위 그림과 같이 filter(kernel)이 합성곱을 통해서 Feature Map을 출력한다.

Stride
그럼 이 filter(kernel)은 위 그림과 같이 우측 상단만 계산을 하고 끝나는가?
아니다. 처음 시작은 좌측 상단 부터 시작에서 우측으로 이동 후 우측 끝까지 이동을 하면은 아래로 내려가 합성곱 연산을 한다.
이때 몇 칸을 이동하는지 사용자가 정할 수 있는데 이를 stride라고 부른다.

궁금했던 점
이제 그림을 보시면 합성곱 연산을 하면은 처음 입력 이미지의 크기가 5x5인데 출력 값의 크기가 3x3으로 작아진 것을 볼 수 있다.
이미지의 해상도 자체가 줄어들었으니 정보가 날라간 것이 아닌가?
이에 대한 답 : 정보 소실이 있는 것은 맞다. 하지만 이미지를 판별할 수 있다. 여기서 이미지를 판별하는 것은 고양이 이미지를 예를 들겠다.
각 이미지의 각 픽셀이 고양이와 동일한 가를 비교하는 것이 아닌 고양이의 특징을 가지고 있는지를 비교해야 하는 관점으로 바라봐야 한다.
즉, 입력 이미지는 합성곱 연산을 하면은 피처맵의 크기는 작아지고 달라지지만 해당 이미지의 특징을 압축해서 가지고 있다.

Padding
- 출력 값의 가장자리 테두리를 채움
왜 채우는 가?
- 합성곱 연산을 하다보면 출력값의 해상도 크기가 한 없이 작아지는 현상을 막기 위해서 사용
- 가장자리 정보가 소실되는 현상을 막기 위해서

해상도 크기가 한 없이 작아지면 안되는 이유 : 출력값의 해상도가 한 없이 작아지게 되면 딥러닝의 층을 계속 쌓을 수 없게 된다.

궁금했던 점
Q. Padding이 가장자리 정보 소실을 어떻게 막지?
A. 합성곱 연산을 하다보면 해상도의 크기가 작아진다. 이때 여기서 합성곱을 할 때 중앙 픽셀은 커널이 여러번 지나쳐 연산 횟수가 많아진다.
이는 해당 픽셀의 특징이 풍부하고 강하게 남는다는 것을 의미한다.
하지만 가장자리 픽셀 같은 경우 커널이 중앙픽셀보다 적게 지나치게 된다. 이는 중앙 픽셀은 커널이 여러 번 지나쳐 특징이 풍부하고 강하게 남지만,
가장자리 픽셀은 커널이 지나치는 경우가 중앙픽셀보다 적어 연산량이 적어 특징이 옅어져 가장 자리 정보를 소실하게 된다.
이를 막기 위해서 Padding을 통해 가장자리 정보를 안쪽에 위치하게 하여 커널이 좀 더 지나치게 만들어 연산량을 늘려 정보 소실을 막게 되는 것이다.

Pooling Layer
- 이미지의 크기를 유지한채 fc_layer로 가게 된다면 연산량이 기하급수적으로 많아진다.
- 따라서 Pooling을 사용하여 feature map의 크기를 줄이고 Feature를 강조한다.

Pooling의 이점
- 이미지는 인접 픽셀들 간의 유사도가 높다. 따라서 각각의 픽셀을 개별적으로 처리하기보다는 선택 영역을 통해 처리하면은 연산이 효율적이다.
- 눈동자 이미지가 있다고 가정하자 이미지의 내의 눈동자 픽셀이 약간 움직여도 pooling을 통해 해당 지역 내 특징을 뽑기 때문에 이미지 내의 객체 이동 및 회전 문제를 완화할 수 있다.
- pooling을 통해 parameter수를 줄일 수 있고 이를 통해 overfiting문제도 완화시킨다.

FC_layer(Fully Connected Layer)
위의 방식을 통해 나온 피처 맵을 통해 이것이 무엇인지 의미하는 데이터인지 분류

설정 가능한 파라미터 수
Convolution filter 개수
- CNN에서는 Convolution layer를 거친 후에 Pooling layer를 지나가게 되는데 이때 Convolution layer를 통해서 나온 피처 맵의 해상도 크기가 1/4로 줄어들게 된다.(pooling:kernel 2, stride 2)
  이때 Pooling layer로 정보를 압축한다고 하지만 정보가 소실될 수 있다. 따라서 filter 개수를 늘려 채널 수를 깊게 만들어 다양하고 복잡한 정보를 담을 수 있게 한다.
Filter size
- 보통 3x3사이즈 사용
Padding 여부
- 출력 해상도의 크기가 줄어드는 것을 방지
Stride
- 필터를 이용하여 Convolution할 때 몇 칸을 기준으로 하는지 설정
Pooling layer
- max Pooling, avg Pooling등 종류를 선택
